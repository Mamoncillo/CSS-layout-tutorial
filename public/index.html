position: static vs relative vs absolute

css info

Position property

The position: CSS property specifies the type of positioning method used for an element (static, relative, absolute or fixed).

Look at css tag and follow the instructions for best understanding of the information here

Static

Static is the default value. Elements render in order as they appear in the html flow, the top, right, bottom, left and z-index properties do not apply.

Relative

The element is positioned relative to its normal position, in simple words, it is like static, elements render in order as they appear in the html document, but you can move it around using left, right, top and bottom relative to its position, it will leave a white space where it is supposed to be in first place.

Absolute

The element is positioned relative to its first positioned (not static) ancestor element, it will not leave white space for the element

If you want to have more control over absolute elements position, put them inside a non static element

Fixed

Do not leave space for the element. Instead, position it at a specified position relative to the screen's viewport and don't move it when scrolled.
